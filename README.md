# Python-calculator
A simple Python calculator application that performs basic arithmetic operations using menu-driven interface.

# Features
1)Addition
2)Subtraction
3)Multiplication
4)Division
5)Power
6)Divison by Zero Handling
7)Calculation possible until exit selected

# Technologies Used
Python 3

# Concepts Used
1)Functions
2)Loops
3)Conditional Statements
4)User Input
5)Error Handling
